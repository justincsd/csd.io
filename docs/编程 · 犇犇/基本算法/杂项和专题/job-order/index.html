在一台机器上规划任务

你有 nnn 个任务，要求你找到一个代价最小的的顺序执行他们。第 iii 个任务花费的时间是 tit_iti​，而第 iii 个任务等待 ttt 的时间会花费 fi(t)f_i(t)fi​(t) 的代价。

形式化地说，给出 nnn 个函数 fif_ifi​ 和 nnn 个数 tit_iti​，求一个排列 ppp，最小化

F(p)=∑i=1nfpi(∑j=1i−1tpj)F(p)=\sum_{i=1}^nf_{p_i}\left(\sum_{j=1}^{i-1}t_{p_j}\right)F(p)=i=1∑n​fpi​​(j=1∑i−1​tpj​​)

特殊的代价函数​

线性代价函数​

首先我们考虑所有的函数是线性的函数，即 fi(x)=cix+dif_i(x)=c_ix+d_ifi​(x)=ci​x+di​，其中 cic_ici​ 是非负整数。显然我们可以事先把常数项加起来，因此函数就转化为了 fi(x)=cixf_i(x)=c_ixfi​(x)=ci​x 的形式。

考虑两个排列 ppp 和 p′p'p′，其中 p′p'p′ 是把 ppp 的第 iii 个位置上的数和 i+1i+1i+1 个位置上的数交换得到的排列。则

F(p′)−F(p)=cpi′∑j=1i−1tpj′+cpi+1′∑j=1itpj′−(cpi∑j=1i−1tpj+cpi+1∑j=1itpj)=cpitpi+1−cpi+1tpi\begin{aligned} F(p')-F(p)&=c_{p'_i}\sum_{j=1}^{i-1}t_{p'_j}+c_{p'_{i+1}}\sum_{j=1}^{i}t_{p'_j} -\left(c_{p_i}\sum_{j=1}^{i-1}t_{p_j}+c_{p_{i+1}}\sum_{j=1}^{i}t_{p_j}\right)\\ &=c_{p_i}t_{p_{i+1}}-c_{p_{i+1}}t_{p_i} \end{aligned}F(p′)−F(p)​=cpi′​​j=1∑i−1​tpj′​​+cpi+1′​​j=1∑i​tpj′​​−(cpi​​j=1∑i−1​tpj​​+cpi+1​​j=1∑i​tpj​​)=cpi​​tpi+1​​−cpi+1​​tpi​​​

于是我们使用如果 cpitpi+1−cpi+1tpi>0c_{p_i}t_{p_{i+1}}-c_{p_{i+1}}t_{p_i}>0cpi​​tpi+1​​−cpi+1​​tpi​​>0 就交换的策略做一下排序就可以了。写成 cpitpi>cpi+1tpi+1\dfrac{c_{p_i}}{t_{p_i}}>\dfrac{c_{p_{i+1}}}{t_{p_{i+1}}}tpi​​cpi​​​>tpi+1​​cpi+1​​​ 的形式，就可以理解为将排列按 citi\dfrac{c_i}{t_i}ti​ci​​ 升序排序。

处理这个问题，我们的思路是考虑微扰后的变换情况，贪心地选取最优解。

指数代价函数​

考虑代价函数的形式为 fi(x)=cieaxf_i(x)=c_i\mathrm{e}^{ax}fi​(x)=ci​eax，其中 ci≥0,a>0c_i\ge 0,a>0ci​≥0,a>0。

我们沿用之前的思路，考虑将 iii 和 i+1i+1i+1 的位置上的数交换引起的代价变化。最终得到的算法是将排列按照 1−eatici\dfrac{1-\mathrm{e}^{at_i}}{c_i}ci​1−eati​​ 升序排序。

相同的单增函数​

我们考虑所有的 fi(x)f_i(x)fi​(x) 是同一个单增函数。那么显然我们将排列按照 tit_iti​ 升序排序即可。

Livshits–Kladov 定理​

Livshits–Kladov 定理成立，当且仅当代价函数是以下三种情况：

  • 线性函数：fi(t)=cit+dif_i(t) = c_it + d_ifi​(t)=ci​t+di​，其中 ci≥0c_i\ge 0ci​≥0；
  • 指数函数：fi(t)=cieat+dif_i(t) = c_i \mathrm{e}^{a t} + d_ifi​(t)=ci​eat+di​，其中 ci,a>0c_i,a>0ci​,a>0；
  • 相同的单增函数：fi(t)=ϕ(t)f_i(t) = \phi(t)fi​(t)=ϕ(t)，其中 ϕ(t)\phi(t)ϕ(t) 是一个单增函数。

定理是在假设代价函数足够平滑（存在三阶导数）的条件下证明的。在这三种情况下，问题的最优解可以通过简单的排序在 O(nlog⁡n)O(n\log n)O(nlogn) 的时间内解决。
