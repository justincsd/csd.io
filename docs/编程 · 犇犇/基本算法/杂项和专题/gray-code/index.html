格雷码

格雷码是一个二进制数系，其中两个相邻数的二进制位只有一位不同。举个例子，333 位二进制数的格雷码序列为

000,001,011,010,110,111,101,100000,001,011,010,110,111,101,100000,001,011,010,110,111,101,100

注意序列的下标我们以 000 为起点，也就是说 G(0)=000,G(4)=110G(0)=000,G(4)=110G(0)=000,G(4)=110。

格雷码由贝尔实验室的 Frank Gray 于 1940 年代提出，并于 1953 年获得专利。

构造格雷码（变换）​

格雷码的构造方法很多。我们首先介绍手动构造方法，然后会给出构造的代码以及正确性证明。

手动构造​

kkk 位的格雷码可以通过以下方法构造。我们从全 000 格雷码开始，按照下面策略：

  1. 翻转最低位得到下一个格雷码，（例如 000→001000\to 001000→001）；
  2. 把最右边的 111 的左边的位翻转得到下一个格雷码，（例如 001→011001\to 011001→011）；

交替按照上述策略生成 2k−12^{k-1}2k−1 次，可得到 kkk 位的格雷码序列。

计算方法​

我们观察一下 nnn 的二进制和 G(n)G(n)G(n)。可以发现，如果 G(n)G(n)G(n) 的二进制第 iii 位为 111，仅当 nnn 的二进制第 iii 位为 111，第 i+1i+1i+1 位为 000 或者第 iii 位为 000，第 i+1i+1i+1 位为 111。于是我们可以当成一个异或的运算，即

G(n)=n⊕⌊n2⌋G(n)=n\oplus \left\lfloor\frac{n}{2}\right\rfloorG(n)=n⊕⌊2n​⌋ 
int g(int n) { return n ^ (n >> 1); }

正确性证明​

接下来我们证明一下，按照上述公式生成的格雷码序列，相邻两个格雷码的二进制位有且仅有一位不同。

我们考虑 nnn 和 n+1n+1n+1 的区别。把 nnn 加 111，相当于把 nnn 的二进制下末位的连续的 111 全部变成取反，然后把最低位的 000 变成 111。我们这样表示 nnn 和 n+1n+1n+1 的二进制位：

(n)2=⋯011⋯11⏟k个(n+1)2=⋯100⋯00⏟k个\begin{aligned} (n)_2 &= \cdots0\underbrace{11\cdots11}_{k\text{个}}\\ (n+1)_2 &= \cdots1\underbrace{00\cdots00}_{k\text{个}} \end{aligned}(n)2​(n+1)2​​=⋯0k个11⋯11​​=⋯1k个00⋯00​​​

于是我们在计算 g(n)g(n)g(n) 和 g(n+1)g(n+1)g(n+1) 的时侯，后 kkk 位都会变成 100⋯00⏟k个\displaystyle\underbrace{100\cdots00}_{k\text{个}}k个100⋯00​​ 的形式，而第 k+1k+1k+1 位是不同的，因为 nnn 和 n+1n+1n+1 除了后 k+1k+1k+1 位，其他位都是相同的。因此第 k+1k+1k+1 位要么同时异或 111，要么同时异或 000。两种情况，第 k+1k+1k+1 位都是不同的。而除了后 k+1k+1k+1 位，其他位都是相同的，因此异或后都是 000。

证毕。

实际应用​

格雷码有一些十分有用的应用，有些应用让人意想不到：

  • kkk 位二进制数的格雷码序列可以当作 kkk 维空间中的一个超立方体（二维里的正方形，一维里的单位向量）顶点的哈密尔顿回路，其中格雷码的每一位代表一个维度的坐标。

  • 格雷码被用于最小化数字模拟转换器（比如传感器）的信号传输中出现的错误，因为它每次只改变一个位。

  • 格雷码可以用来解决汉诺塔的问题。

    设盘的数量为 nnn。我们从 nnn 位全 000 的格雷码 G(0)G(0)G(0) 开始，依次移向下一个格雷码（G(i)G(i)G(i) 移向 G(i+1)G(i+1)G(i+1)）。当前格雷码的二进制第 iii 位表示从小到大第 iii 个盘子。

    由于每一次只有一个二进制位会改变，因此当第 iii 位改变时，我们移动第 iii 个盘子。在移动盘子的过程中，除了最小的盘子，其他任意一个盘子在移动的时侯，只能有一个放置选择。在移动第一个盘子的时侯，我们总是有两个放置选择。于是我们的策略如下：

    如果 nnn 是一个奇数，那么盘子的移动路径为 f→t→r→f→t→r→⋯f\to t\to r\to f\to t\to r\to\cdotsf→t→r→f→t→r→⋯，其中 fff 是最开始的柱子，ttt 是最终我们把所有盘子放到的柱子，rrr 是中间的柱子。

    如果 nnn 是偶数：f→r→t→f→r→t→⋯f \to r \to t \to f \to r \to t \to \cdotsf→r→t→f→r→t→⋯

  • 格雷码也在遗传算法理论中得到应用。
