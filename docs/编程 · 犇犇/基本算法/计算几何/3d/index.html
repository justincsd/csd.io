三维计算几何 · 基础

三维几何的很多概念与 二维几何 是相通的，我们可以用与解决二维几何问题相同的方法来解决三维几何问题。

基本概念​

点，向量，直线这些概念和二维几何是相似的，这里不再展开。

平面​

我们可以用平面上的一点 P0(x0,y0,z0)P_0(x_0,y_0,z_0)P0​(x0​,y0​,z0​) 和该平面的法向量（即垂直于该平面的向量）n\boldsymbol{n}n 来表示一个平面。

因为 n\boldsymbol{n}n 垂直于平面，所以 n\boldsymbol{n}n 垂直于该平面内的所有直线。换句话说，设 n=(A,B,C)\boldsymbol{n}=(A,B,C)n=(A,B,C)，则该平面上的点 P(x,y,z)P(x,y,z)P(x,y,z) 都满足 n⋅PP0→=0\boldsymbol{n} \cdot \overrightarrow{PP_0} = 0n⋅PP0​​=0。

根据向量点积的定义，上式等价于：

A(x−x0)+B(y−y0)+C(z−z0)=0A(x-x_0)+B(y-y_0)+C(z-z_0)=0A(x−x0​)+B(y−y0​)+C(z−z0​)=0

整理后得到：

Ax+By+Cz−(Ax0+By0+Cz0)=0Ax+By+Cz-(Ax_0+By_0+Cz_0)=0Ax+By+Cz−(Ax0​+By0​+Cz0​)=0

令 D=−(Ax0+By0+Cz0)D=-(Ax_0+By_0+Cz_0)D=−(Ax0​+By0​+Cz0​)，则上式变成 Ax+By+Cz+D=0Ax+By+Cz+D=0Ax+By+Cz+D=0。我们称这个式子为平面的 一般式。

基本操作​

直线、平面之间的夹角​

运用空间向量的知识，空间中直线、平面之间的夹角可以很快求出。

对于两条异面直线 aaa，bbb，过空间中一点 PPP，作 a′∥aa' \parallel aa′∥a，b′∥b′b' \parallel b'b′∥b′，则 a′a'a′ 与 b′b'b′ 所成的锐角或直角被称为 aaa 和 bbb 两条 异面直线所成的角。

对于直线 aaa 和平面 α\alphaα，若 aaa 与 α\alphaα 相交于 AAA，过 aaa 上一点 PPP 引平面 α\alphaα 的垂线交 α\alphaα 于 OOO，则 aaa 与 POPOPO 所成的角被称为 直线与平面所成的角。特别地，若 a∥αa \parallel \alphaa∥α 或 a⊂αa \subset \alphaa⊂α，则它们之间的夹角为 0∘0^\circ0∘。

对于两个平面 α\alphaα，β\betaβ，它们的夹角被定义为与两条平面的交线 lll 垂直的两条直线 a,ba,ba,b（其中 a⊂αa \subset \alphaa⊂α，b⊂βb \subset \betab⊂β）所成的角。
