校董代表大会 SMM

  1. 校董代表大会是校董协会的最高权力机关，具有至高无上的立法权、指挥权、解释权等。
  2. 校董代表大会是重要事情的决策方案机构，任何有关政协的决定先在校董代表大会进行。
  3. 校董代表大会又称 SMM，也就是 School Managers` Association，请注意言行举止。
  4. 校董代表大会基于 SMA 建立，是核心的会议和权力机关。

决策信息

定期对管理层面展开调查和抽查，随时可以查看管理层级的工作态度。

重要信息

任何决定由当时情况决定，以校长的决策为重要目标，但是不要搞个人崇拜。
