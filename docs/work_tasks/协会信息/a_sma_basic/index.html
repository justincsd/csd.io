注意 / 管理员入职条件。

如果有任何疑问，请联系上述管理员，暂时不支持主动申请管理员职位，需要得到认可与邀请。

目前有以下条件：

  1. 对集体有比较大的贡献，包括但不限于制度创新然后越走越好；
  2. 有比较特殊的身份，包括但不限于天生就是乐子（ZYZ 不就是吗）；
  3. 拥有比较充裕的时间，对于集体机制熟悉。
