欧拉函数（Euler）

定义​

欧拉函数（Euler's totient function），即 φ(n)\varphi(n)φ(n)，表示的是小于等于 nnn 和 nnn 互质的数的个数。

比如说 φ(1)=1\varphi(1) = 1φ(1)=1。

当 nnn 是质数的时候，显然有 φ(n)=n−1\varphi(n) = n - 1φ(n)=n−1。

性质​

  • 欧拉函数是积性函数。

    即对任意满足 gcd⁡(a,b)=1\gcd(a, b) = 1gcd(a,b)=1 的整数 a,ba,ba,b，有 φ(ab)=φ(a)φ(b)\varphi(ab) = \varphi(a)\varphi(b)φ(ab)=φ(a)φ(b)。

    特别地，当 nnn 是奇数时 φ(2n)=φ(n)\varphi(2n) = \varphi(n)φ(2n)=φ(n)。

  • n=∑d∣nφ(d)n = \sum_{d \mid n}{\varphi(d)}n=∑d∣n​φ(d)。

    证明

    利用莫比乌斯反演相关知识可以得出。

    也可以这样考虑：如果 gcd⁡(k,n)=d\gcd(k, n) = dgcd(k,n)=d，那么 gcd⁡(kd,nd)=1,(k<n)\gcd(\dfrac{k}{d},\dfrac{n}{d}) = 1, ( k < n )gcd(dk​,dn​)=1,(k<n)。

    如果我们设 f(x)f(x)f(x) 表示 gcd⁡(k,n)=x\gcd(k, n) = xgcd(k,n)=x 的数的个数，那么 n=∑i=1nf(i)n = \sum_{i = 1}^n{f(i)}n=∑i=1n​f(i)。

    根据上面的证明，我们发现，f(x)=φ(nx)f(x) = \varphi(\dfrac{n}{x})f(x)=φ(xn​)，从而 n=∑d∣nφ(nd)n = \sum_{d \mid n}\varphi(\dfrac{n}{d})n=∑d∣n​φ(dn​)。注意到约数 ddd 和 nd\dfrac{n}{d}dn​ 具有对称性，所以上式化为 n=∑d∣nφ(d)n = \sum_{d \mid n}\varphi(d)n=∑d∣n​φ(d)。

  • 若 n=pkn = p^kn=pk，其中 ppp 是质数，那么 φ(n)=pk−pk−1\varphi(n) = p^k - p^{k - 1}φ(n)=pk−pk−1。 （根据定义可知）

  • 由唯一分解定理，设 n=∏i=1spikin = \prod_{i=1}^{s}p_i^{k_i}n=∏i=1s​piki​​，其中 pip_ipi​ 是质数，有 φ(n)=n×∏i=1spi−1pi\varphi(n) = n \times \prod_{i = 1}^s{\dfrac{p_i - 1}{p_i}}φ(n)=n×∏i=1s​pi​pi​−1​。

  • 由唯一分解定理，设 n=∏i=1spikin = \prod_{i=1}^{s}p_i^{k_i}n=∏i=1s​piki​​，其中 pip_ipi​ 是质数，有 φ(n)=n×∏i=1spi−1pi\varphi(n) = n \times \prod_{i = 1}^s{\dfrac{p_i - 1}{p_i}}φ(n)=n×∏i=1s​pi​pi​−1​。

    证明

    • 引理：设 ppp 为任意质数，那么 φ(pk)=pk−1×(p−1)\varphi(p^k)=p^{k-1}\times(p-1)φ(pk)=pk−1×(p−1)。

      证明：显然对于从 1 到 pkp^kpk 的所有数中，除了 pk−1p^{k-1}pk−1 个 ppp 的倍数以外其它数都与 pkp^kpk 互素，故 φ(pk)=pk−pk−1=pk−1×(p−1)\varphi(p^k)=p^k-p^{k-1}=p^{k-1}\times(p-1)φ(pk)=pk−pk−1=pk−1×(p−1)，证毕。

      接下来我们证明 φ(n)=n×∏i=1spi−1pi\varphi(n) = n \times \prod_{i = 1}^s{\dfrac{p_i - 1}{p_i}}φ(n)=n×∏i=1s​pi​pi​−1​。由唯一分解定理与 φ(x)\varphi(x)φ(x) 函数的积性

      φ(n)=∏i=1sφ(piki)=∏i=1s(pi−1)×piki−1=∏i=1spiki×(1−1pi)=n ∏i=1s(1−1pi)□\begin{aligned} \varphi(n) &= \prod_{i=1}^{s} \varphi(p_i^{k_i}) \\ &= \prod_{i=1}^{s} (p_i-1)\times {p_i}^{k_i-1}\\ &=\prod_{i=1}^{s} {p_i}^{k_i} \times(1 - \frac{1}{p_i})\\ &=n~ \prod_{i=1}^{s} (1- \frac{1}{p_i}) &\square \end{aligned}φ(n)​=i=1∏s​φ(piki​​)=i=1∏s​(pi​−1)×pi​ki​−1=i=1∏s​pi​ki​×(1−pi​1​)=n i=1∏s​(1−pi​1​)​□​

    可由上一条直接计算得出。

实现​

如果只要求一个数的欧拉函数值，那么直接根据定义质因数分解的同时求就好了。这个过程可以用 Pollard Rho 算法优化。

#include <cmath>

int euler_phi(int n) {
  int ans = n;
  for (int i = 2; i * i <= n; i++)
    if (n % i == 0) {
      ans = ans / i * (i - 1);
      while (n % i == 0) n /= i;
    }
  if (n > 1) ans = ans / n * (n - 1);
  return ans;
}

如果是多个数的欧拉函数值，可以利用后面会提到的线性筛法来求得。

应用​

欧拉函数常常用于化简一列最大公约数的和。国内有些文章称它为 欧拉反演[^1]。

在结论

n=∑d∣nφ(d)n=\sum_{d|n}\varphi(d)n=d∣n∑​φ(d)

中代入 n=gcd⁡(a,b)n=\gcd(a,b)n=gcd(a,b)，则有

gcd⁡(a,b)=∑d∣gcd⁡(a,b)φ(d)=∑d[d∣a][d∣b]φ(d),\gcd(a,b) = \sum_{d|\gcd(a,b)}\varphi(d) = \sum_d [d|a][d|b]\varphi(d),gcd(a,b)=d∣gcd(a,b)∑​φ(d)=d∑​[d∣a][d∣b]φ(d),

其中，[⋅][\cdot][⋅] 称为 Iverson 括号，只有当命题 PPP 为真时 [P][P][P] 取值为 111，否则取 000。对上式求和，就可以得到

∑i=1ngcd⁡(i,n)=∑d∑i=1n[d∣i][d∣n]φ(d)=∑d⌊nd⌋[d∣n]φ(d)=∑d∣n⌊nd⌋φ(d).\sum_{i=1}^n\gcd(i,n)=\sum_{d}\sum_{i=1}^n[d|i][d|n]\varphi(d)=\sum_d\left\lfloor\frac{n}{d}\right\rfloor[d|n]\varphi(d)=\sum_{d|n}\left\lfloor\frac{n}{d}\right\rfloor\varphi(d).i=1∑n​gcd(i,n)=d∑​i=1∑n​[d∣i][d∣n]φ(d)=d∑​⌊dn​⌋[d∣n]φ(d)=d∣n∑​⌊dn​⌋φ(d).

这里关键的观察是 ∑i=1n[d∣i]=⌊nd⌋\sum_{i=1}^n[d|i]=\lfloor\frac{n}{d}\rfloor∑i=1n​[d∣i]=⌊dn​⌋，即在 111 和 nnn 之间能够被 ddd 整除的 iii 的个数是 ⌊nd⌋\lfloor\frac{n}{d}\rfloor⌊dn​⌋。

利用这个式子，就可以遍历约数求和了。需要多组查询的时候，可以预处理欧拉函数的前缀和，利用数论分块查询。

欧拉定理​

与欧拉函数紧密相关的一个定理就是欧拉定理。其描述如下：

若 gcd⁡(a,m)=1\gcd(a, m) = 1gcd(a,m)=1，则 aφ(m)≡1(modm)a^{\varphi(m)} \equiv 1 \pmod{m}aφ(m)≡1(modm)。
