傅里叶-莫茨金消元法

引入​

傅里叶——莫茨金消元法（原名 Fourier–Motzkin Elimination，简称 FME 算法）是一种用于从线性不等式中消除变量的数学方法。

它的命名源自于在 1827 年和 1936 年独立发现该算法的 Joseph Fourier 和 Theodore Motzkin 的姓氏。

过程​

从线性不等式中消除一组变量，是指通过将关系式中的若干个元素有限次地变换，消去其中的某些元素，从而解决问题的一种方法。

如果线性不等式中的所有变量都被消除，那么我们会得到一个常不等式。因为当且仅当原不等式有解时，消元后的不等式才为真，消除所有变量可用于检测不等式系统是否有解。

考虑一个含 nnn 个不等式的系统 SSS，有从 x1x_{1}x1​ 到 xrx_{r}xr​ 的 rrr 个变量，其中 xrx_{r}xr​ 为要消除的变量。根据 xrx_rxr​ 系数的符号（正、负或空），SSS 中的线性不等式可以分为三类：

  1. 形式为 xr≥bi−∑k=1r−1aikxkx_{r}\geq b_{i}-\sum _{k=1}^{r-1}a_{ik}x_{k}xr​≥bi​−∑k=1r−1​aik​xk​ 的不等式，对于范围从 111 到 nAn_{A}nA​（nAn_{A}nA​ 为这种不等式的数量）的 jjj，用 xr≥Aj(x1,…,xr−1)x_{r}\geq A_{j}(x_{1},\dots ,x_{r-1})xr​≥Aj​(x1​,…,xr−1​) 表示；
  2. 形式为 xr≤bi−∑k=1r−1aikxkx_{r}\leq b_{i}-\sum _{k=1}^{r-1}a_{ik}x_{k}xr​≤bi​−∑k=1r−1​aik​xk​ 的不等式，对于范围从 111 到 nBn_{B}nB​（nBn_{B}nB​ 为这种不等式的数量）的 jjj，用 xr≤Bj(x1,…,xr−1)x_{r}\leq B_{j}(x_{1},\dots ,x_{r-1})xr​≤Bj​(x1​,…,xr−1​) 表示；
  3. 不包含 xrx_{r}xr​ 的不等式，设它们构成的不等式组为 ϕ\phiϕ。

因此原系统等价于

max⁡(A1(x1,…,xr−1),…,AnA(x1,…,xr−1))≤xr≤min⁡(B1(x1,…,xr−1),…,BnB(x1,…,xr−1))∧ϕ\max(A_{1}(x_{1},\dots ,x_{r-1}),\dots ,A_{n_{A}}(x_{1},\dots ,x_{r-1}))\leq x_{r}\leq \min(B_{1}(x_{1},\dots ,x_{r-1}),\dots ,B_{n_{B}}(x_{1},\dots ,x_{r-1}))\wedge \phimax(A1​(x1​,…,xr−1​),…,AnA​​(x1​,…,xr−1​))≤xr​≤min(B1​(x1​,…,xr−1​),…,BnB​​(x1​,…,xr−1​))∧ϕ

消元包括产生一个等价于 ∃xr S\exists x_{r}~S∃xr​ S 的系统。显然，这个公式等价于

max⁡(A1(x1,…,xr−1),…,AnA(x1,…,xr−1))≤min⁡(B1(x1,…,xr−1),…,BnB(x1,…,xr−1))∧ϕ\max(A_{1}(x_{1},\dots ,x_{r-1}),\dots ,A_{n_{A}}(x_{1},\dots ,x_{r-1}))\leq \min(B_{1}(x_{1},\dots ,x_{r-1}),\dots ,B_{n_{B}}(x_{1},\dots ,x_{r-1}))\wedge \phimax(A1​(x1​,…,xr−1​),…,AnA​​(x1​,…,xr−1​))≤min(B1​(x1​,…,xr−1​),…,BnB​​(x1​,…,xr−1​))∧ϕ

不等式

max⁡(A1(x1,…,xr−1),…,AnA(x1,…,xr−1))≤min⁡(B1(x1,…,xr−1),…,BnB(x1,…,xr−1))\max(A_{1}(x_{1},\dots ,x_{r-1}),\dots ,A_{n_{A}}(x_{1},\dots ,x_{r-1}))\leq \min(B_{1}(x_{1},\dots ,x_{r-1}),\dots ,B_{n_{B}}(x_{1},\dots ,x_{r-1}))max(A1​(x1​,…,xr−1​),…,AnA​​(x1​,…,xr−1​))≤min(B1​(x1​,…,xr−1​),…,BnB​​(x1​,…,xr−1​))

等价于对于 1≤i≤nA1 \leq i \leq n_{A}1≤i≤nA​ 且 1≤j≤nB1\leq j\leq n_{B}1≤j≤nB​，所有 nAnBn_{A}n_{B}nA​nB​ 个不等式 Ai(x1,…,xr−1)≤Bj(x1,…,xr−1)A_{i}(x_{1},\dots ,x_{r-1})\leq B_{j}(x_{1},\dots ,x_{r-1})Ai​(x1​,…,xr−1​)≤Bj​(x1​,…,xr−1​) 构成的不等式组。

因此，我们将原系统 SSS 转换为另一个消掉 xrx_{r}xr​ 的系统，这个系统有 (n−nA−nB)+nAnB(n-n_{A}-n_{B})+n_{A}n_{B}(n−nA​−nB​)+nA​nB​ 个不等式。特别地，如果 nA=nB=n/2n_{A}=n_{B}=n/2nA​=nB​=n/2，那么新系统不等式的个数为 n2/4n^{2}/4n2/4。

例题​

考虑以下不等式系统：

2x−5y+4z≤102x-5y+4z \leq 102x−5y+4z≤10

3x−6y+3z≤93x-6y+3z \leq 93x−6y+3z≤9

−x+5y−2z≤−7-x+5y-2z \leq -7−x+5y−2z≤−7

−3x+2y+6z≤12-3x+2y+6z \leq 12−3x+2y+6z≤12

为了消除 xxx，我们可以根据 xxx 改写不等式：

x≤(10+5y−4z)/2x \leq (10 + 5y - 4z)/2x≤(10+5y−4z)/2

x≤(9+6y−3z)/3x \leq (9 + 6y - 3z)/3x≤(9+6y−3z)/3

x≥7+5y−2zx \geq 7 + 5y - 2zx≥7+5y−2z

x≥(−12+2y+6z)/3x \geq (-12 + 2y + 6z)/3x≥(−12+2y+6z)/3

这样我们得到两个 ≤\leq≤ 不等式和两个 ≥\geq≥ 不等式；如果每个 ≤\leq≤ 不等式的右侧至少是每个 ≥\geq≥ 不等式的右侧，则系统有一个解。我们有 2×22\times22×2 这样的组合：

7+5y−2z≤(10+5y−4z)/27 + 5y - 2z \leq (10 + 5y - 4z)/27+5y−2z≤(10+5y−4z)/2

7+5y−2z≤(9+6y−3z)/37 + 5y - 2z \leq (9 + 6y - 3z)/37+5y−2z≤(9+6y−3z)/3

(−12+2y+6z)/3≤(10+5y−4z)/2(-12 + 2y + 6z)/3 \leq (10 + 5y - 4z)/2(−12+2y+6z)/3≤(10+5y−4z)/2

(−12+2y+6z)/3≤(9+6y−3z)/3(-12 + 2y + 6z)/3 \leq (9 + 6y - 3z)/3(−12+2y+6z)/3≤(9+6y−3z)/3

现在我们有了一个新的少了一个变量不等式系统。

时间复杂度​

在 nnn 个不等式上消元可以最多得到 n2/4n^{2}/4n2/4 个不等式，因此连续运行 ddd 步可以得到最多 4(n/4)2d4(n/4)^{2^{d}}4(n/4)2d 的双指数复杂度。这是由于算法产生了许多不必要的约束（其他约束隐含的约束）。必要约束的数量以单一指数增长。

可以使用线性规划 (Linear Programming, LP) 检测不必要的约束。

应用​

信息论的可实现性证明保证了存在性能良好的编码方案的条件。这些条件通常使用线性不等式系统描述。系统的变量包括传输速率和附加辅助速率。通常，人们旨在仅根据问题的参数（即传输速率）来描述通信的基本限制，因此述辅助率需要消除上。而我们正是通过傅立叶 - 莫茨金消元法来做到这一点的。

实现​

在编程语言中，Racket，一种基于 Lisp 的多范式编程语言在 fme - Fourier–Motzkin Elimination for Integer Systems) 中对 FME 算法做了简单函数代数实现。
