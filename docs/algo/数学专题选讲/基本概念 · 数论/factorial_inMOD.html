阶乘取模

引入​

本文讨论了某一模数下阶乘计算的相关结论，并提供一种时间复杂度线性相关于模数大小的计算方法，因而该方法主要适用于模数不太大（∼106\sim 10^6∼106）的情形。除了本文介绍的方法外，根据场景不同，还可以应用多项式技术进行快速计算。

根据 CRT，阶乘取模问题可以转化为模数为素数幂 pαp^\alphapα 的情形。在处理这类问题时，常常需要对于素数 ppp 和正整数 nnn，将阶乘 n!n!n! 中的所有因子 ppp 都提取出来，进而得到分解：

n!=pνp(n!)(n!)p.n! = p^{\nu_p(n!)}(n!)_p.n!=pνp​(n!)(n!)p​.

其中，νp(n!)\nu_p(n!)νp​(n!) 表示阶乘 n!n!n! 的素因数分解中 ppp 的幂次，(n!)p(n!)_p(n!)p​ 表示在阶乘 n!n!n! 的结果中去除所有 ppp 的幂次得到的整数。本文将讨论 (n!)p(n!)_p(n!)p​ 在素数（幂）模下的余数以及幂次 νp(n!)\nu_p(n!)νp​(n!) 的具体计算方法。

这种分解在解决阶乘同时出现在所求表达式的分子和分母的问题时尤为有用，比如计算某一模数下的二项式系数。对于这类问题，分子和分母中 ppp 的幂次可以直接相减，而与 ppp 互素的部分 (n!)p(n!)_p(n!)p​ 则可以利用乘法逆元计算。

本文还介绍了与上述问题相关的 Wilson 定理及其推广、Legendre 公式和 Kummer 定理等内容。

Wilson 定理​

Wilson 定理给出了判断某个自然数是素数的一个充分必要条件。

Wilson 定理

对于自然数 n>1n>1n>1，当且仅当 nnn 是素数时，(n−1)!≡−1(modn)(n-1)!\equiv -1\pmod n(n−1)!≡−1(modn)。

首先，证明对于素数 ppp 有 (p−1)!≡−1(modp)(p-1)!\equiv -1\pmod{p}(p−1)!≡−1(modp)。对于这一点，可以利用同余方程或原根得到两种简洁的证明，此处略去不表。下面提供前置知识较少的一种证明方法：

当 p=2p=2p=2 时，命题显然成立。下面设 p≥3p\geq 3p≥3，继而要证明 Zp\mathbf{Z}_pZp​ 中所有非零元素（即同余类）的积为 −1‾\overline{-1}−1​。因为 Zp\mathbf{Z}_pZp​ 中所有非零元素 a‾\overline{a}a 都有逆元 a‾−1\overline{a}^{-1}a−1，于是 Zp\mathbf{Z}_pZp​ 中彼此互逆的元素乘积为 1‾\overline{1}1。但是要注意 a‾\overline{a}a 和 a‾−1\overline{a}^{-1}a−1 可能相等：a‾=a‾−1\overline{a}=\overline{a}^{-1}a=a−1，当且仅当 a2≡1(modp)a^2\equiv 1\pmod pa2≡1(modp)，即

0≡a2−1≡(a+1)(a−1),(modp)0\equiv a^2-1\equiv (a+1)(a-1),\pmod p0≡a2−1≡(a+1)(a−1),(modp)

从而，a≡1(modp)a\equiv 1\pmod pa≡1(modp) 或 a≡−1(modp)a\equiv -1\pmod pa≡−1(modp)。这说明 Zp∖{0‾,1‾,−1‾}\mathbf{Z}_p\setminus\{\overline{0},\overline{1},\overline{-1}\}Zp​∖{0,1,−1​} 中所有元素的乘积为 1‾\overline{1}1，进而 Zp\mathbf{Z}_pZp​ 中所有非零元素的积为 −1‾\overline{-1}−1​。

反过来，对于合数 nnn 的情形，要证明 (n−1)!≢−1(modn)(n-1)!\not\equiv-1\pmod{n}(n−1)!≡−1(modn)。利用反证法，不妨设 (n−1)!≡−1(modn)(n-1)!\equiv -1\pmod{n}(n−1)!≡−1(modn)，亦即存在整数 kkk 使得 (n−1)!=kn−1(n-1)!=kn-1(n−1)!=kn−1 成立。因为 nnn 是合数，必然存在素数 p<np<np<n 使得 n=pmn=pmn=pm，所以 (n−1)!=kpm−1≡−1(modp)(n-1)!=kpm-1\equiv -1\pmod{p}(n−1)!=kpm−1≡−1(modp)。但是，乘积 (n−1)!(n-1)!(n−1)! 中必然已经出现 ppp，故而一定有 (n−1)!≡0(modp)(n-1)!\equiv 0\pmod{p}(n−1)!≡0(modp)。这一矛盾就说明了 (n−1)!≢−1(modn)(n-1)!\not\equiv-1\pmod{n}(n−1)!≡−1(modn)。

利用本文的记号，Wilson 定理可以写作 (p!)p≡−1(modp)(p!)_p\equiv -1\pmod{p}(p!)p​≡−1(modp)。

实现方案
// Calculate (n!)_p mod p.
int factmod(int n, int p) {
  // Pretreatment.
  std::vector<int> f(p);
  f[0] = 1;
  for (int i = 1; i < p; ++i) {
    f[i] = (long long)f[i - 1] * i % p;
  }
  // Recursion.
  int res = 1;
  while (n > 1) {
    if ((n / p) & 1) res = p - res;
    res = (long long)res * f[n % p] % p;
    n /= p;
  }
  return res;
}

如果空间有限，无法存储所有阶乘，也可以将函数调用中实际用到的阶乘 n! mod pn!\bmod pn!modp 中的 nnn 都计算出来，然后对它们进行排序，从而可以在最后一次性计算出来这些阶乘的值，汇总到最终结果中，而避免存储所有阶乘的值。

// Calculate (n!)_p mod pa.
int factmod(int n, int p, int pa) {
  // Pretreatment.
  std::vector<int> f(pa);
  f[0] = 1;
  for (int i = 1; i < pa; ++i) {
    f[i] = i % p ? (long long)f[i - 1] * i % pa : f[i - 1];
  }

  while (n > 1) {

    res = (long long)res * f[n % pa] % pa;
    n /= p;
  }
  return res;
}

素数 ppp 在组合数 (mn)\dbinom{m}{n}(nm​) 中的幂次，恰好是 ppp 进制下 mmm 减掉 nnn 需要借位的次数，亦即

νp((mn))=Sp(n)+Sp(m−n)−Sp(m)p−1.\nu_p\left(\dbinom{m}{n}\right)=\frac{S_p(n)+S_p(m-n)-S_p(m)}{p-1}.νp​((nm​))=p−1Sp​(n)+Sp​(m−n)−Sp​(m)​.

特别地，组合数中 222 的幂次是 ν2((mn))=S2(n)+S2(m−n)−S2(m)\nu_2\left(\dbinom{m}{n}\right)=S_2(n)+S_2(m-n)-S_2(m)ν2​((nm​))=S2​(n)+S2​(m−n)−S2​(m).

证明

首先证明下面的表达式。为此，利用 Legendre 公式，有

νp((mn))=νp(m!)−νp(n!)−nup((m−n)!)=∑i=1∞(⌊mpi⌋−⌊npi⌋−⌊m−npi⌋)=Sp(n)+Sp(m−n)−Sp(m)p−1.\begin{aligned} \nu_p\left(\dbinom{m}{n}\right) &=\nu_p(m!)-\nu_p(n!)-nu_p((m-n)!) &=\sum_{i=1}^\infty\left(\left\lfloor\dfrac{m}{p^i}\right\rfloor-\left\lfloor\dfrac{n}{p^i}\right\rfloor-\left\lfloor\dfrac{m-n}{p^i}\right\rfloor\right)\\ &=\frac{S_p(n)+S_p(m-n)-S_p(m)}{p-1}. \end{aligned}νp​((nm​))​=νp​(m!)−νp​(n!)−nup​((m−n)!)=p−1Sp​(n)+Sp​(m−n)−Sp​(m)​.​=i=1∑∞​(⌊pim​⌋−⌊pin​⌋−⌊pim−n​⌋)​

该表达式可以理解为 ppp 进制下 mmm 减掉 nnn 需要借位的次数。因为如果在计算第 iii 位（最低位下标是 111）时存在不够减需要借位的情况，那么相减的结果中第 iii 位之前的数字 ⌊m−npi⌋\left\lfloor\dfrac{m-n}{p^i}\right\rfloor⌊pim−n​⌋，其实是 mmm 中第 iii 位之前的数字 ⌊mpi⌋\left\lfloor\dfrac{m}{p^i}\right\rfloor⌊pim​⌋，减去一（即借掉的一），再减去 nnn 中第 iii 位之前的数字得到的差值 ⌊npi⌋\left\lfloor\dfrac{n}{p^i}\right\rfloor⌊pin​⌋，所以，差值

⌊mpi⌋−⌊npi⌋−⌊m−npi⌋=1\left\lfloor\dfrac{m}{p^i}\right\rfloor-\left\lfloor\dfrac{n}{p^i}\right\rfloor-\left\lfloor\dfrac{m-n}{p^i}\right\rfloor = 1⌊pim​⌋−⌊pin​⌋−⌊pim−n​⌋=1

当且仅当发生了一次借位；否则，该差值为 000。因此，上述表达式中的求和式就可以理解为借位发生的次数。这就得到了 Kummer 定理的文字表述。

例题 HDU 2973 - YAPTCHA 实现方案
#include <iostream>

constexpr int M = 1e6 + 5, N = 3 * M + 7;

bool not_prime[N];
int sum[M];

int main() {
  for (int i = 2; i < N; ++i)
    if (!not_prime[i])
      for (int j = 2; j * i < N; ++j) not_prime[j * i] = true;
  for (int i = 1; i < M; ++i) sum[i] = sum[i - 1] + !not_prime[3 * i + 7];

  int t;
  std::cin >> t;
  while (t--) {
    int n;
    std::cin >> n;
    std::cout << sum[n] << std::endl;
  }
}
