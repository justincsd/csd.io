随机变量的数字特征

本文将介绍随机变量的期望、方差等数字特征。

期望​

定义​

离散型随机变量​

设离散型随机变量 XXX 的概率分布为 pi=P{X=xi}p_i = P\{ X = x_i \}pi​=P{X=xi​}，若和式

∑xipi\sum x_i p_i∑xi​pi​

绝对收敛，则称其值为 XXX 的 期望，记作 EXEXEX。

连续型随机变量​

设连续型随机变量 XXX 的密度函数为 f(x)f(x)f(x)。若积分

∫Rxf(x)dx\int_{\mathbb{R}} xf(x) \text{d} x∫R​xf(x)dx

绝对收敛，则称其值为 XXX 的 期望，记作 EXEXEX。

统一定义​

设随机变量 XXX 的分布函数为 F(x)F(x)F(x)，若 Stieltjes 积分

∫RxdF(x)\int_{\mathbb{R}} x \text{d} F(x)∫R​xdF(x)

绝对收敛，则称其值为 XXX 的 期望，记作 EXEXEX。

期望不存在的例子

考虑有如下分布的离散型随机变量 XXX

P{X=(−1)k2kk}=12k,k=1,2,⋯P\left\{ X = (-1)^k \frac{2^k}{k} \right\} = \frac{1}{2^k}, \quad k = 1, 2, \cdotsP{X=(−1)kk2k​}=2k1​,k=1,2,⋯

尽管和式 ∑xipi\sum x_i p_i∑xi​pi​ 收敛于 −ln⁡2- \ln 2−ln2，但由于其不是绝对收敛的，故 XXX 的期望不存在。

再考虑有如下密度函数的连续型随机变量 YYY

f(y)=1π⋅11+y2,y∈(−∞,+∞)f(y) = \frac{1}{\pi} \cdot \frac{1}{1 + y^2}, \quad y \in (-\infty, +\infty)f(y)=π1​⋅1+y21​,y∈(−∞,+∞)

容易验证 YYY 的期望也不存在。

期望的性质​

线性性​

若随机变量 X,YX, YX,Y 的期望存在，则

  • 对任意实数 a,ba, ba,b，有 E(aX+b)=a⋅EX+bE(aX + b) = a \cdot EX + bE(aX+b)=a⋅EX+b。
  • E(X+Y)=EX+EYE(X + Y) = EX + EYE(X+Y)=EX+EY。

随机变量乘积的期望​

若随机变量 XXX,YYY 的期望存在且 XXX,YYY 相互独立，则有

E(XY)=EX⋅EYE(XY) = EX \cdot EYE(XY)=EX⋅EY

注意：上述性质中的独立性 并非 必要条件。

反例

考察随机变量 XXX 和 YYY，其中 XXX 服从 [−1,1][-1, 1][−1,1] 上的均匀分布，Y=X2Y = X^2Y=X2。

期望与概率的转化​

对于随机事件 AAA，考虑其示性函数 IAI_AIA​：

IA(ω)={1,ω∈A0,ω∉AI_A(\omega) = \begin{cases} 1, & \omega \in A \\ 0, & \omega \notin A \end{cases}IA​(ω)={1,0,​ω∈Aω∈/A​

根据定义可以求得其期望 EIA=P(A)EI_A = P(A)EIA​=P(A)。这一转化在实际应用中非常常见。

例子

假设对于一个长为 nnn 的序列 {ai}\{ a_i \}{ai​}，其中 aka_kak​ 以 pkp_kpk​ 的概率取 kkk，以 1−pk1 - p_k1−pk​ 的概率取 000。考虑如何求 S=∑i=1naiS = \sum_{i=1}^{n} a_iS=∑i=1n​ai​ 的期望。

如果使用定义直接求，需要求出 SSS 在每个可能取值处的概率，这个计算过程比较繁琐，这里不展开叙述。

另一方面，用 IkI_kIk​ 表示随机事件 ak=ka_k = kak​=k 的示性函数，则有

S=∑k=1nk⋅IkS = \sum_{k=1}^{n} k \cdot I_kS=k=1∑n​k⋅Ik​

进而不难求出

条件分布与条件期望​

我们之前研究过条件概率，类似的也可以提出所谓条件期望的概念。

定义​

对于两个随机变量 XXX,YYY，在已知 Y=yY = yY=y 的条件下 XXX 的概率分布（密度函数）称之为 条件概率分布（条件概率密度），分别记作

P(X=xi∣Y=y)fX∣Y(x∣y)P( X = x_i | Y = y ) \qquad f_{X|Y}(x|y)P(X=xi​∣Y=y)fX∣Y​(x∣y)

在此条件下，XXX 的期望称为 条件期望，记作 E[X∣Y=y]E[X|Y=y]E[X∣Y=y]。

条件期望的性质​

条件期望的诸多性质可由条件概率推知，在此不做赘述。

值得一提的是 E[X∣Y]E[X | Y]E[X∣Y] 一般是随机变量 YYY 的函数，且这个函数通常不是线性的。但实际上有

E[E[X∣Y]]=EXE[E[X|Y]] = EXE[E[X∣Y]]=EX

上式称作 全期望公式。

应用​

例题：[LA 7736]Pocky

题意：有一根长为 LLL 的 Pocky，每次随机折成两段。若右边一段的长度不大于 ddd 则停止，否则对右边一段重复上述过程。求重复次数的期望。

题解

记 f(x)f(x)f(x) 表示长度为 xxx 的期望次数。x≤dx \leq dx≤d 的情形平凡。

当 x>dx > dx>d 时，不妨设折断的位置距右端的长度为 kkk，则显然 k∼U[0,x]k \sim U[0, x]k∼U[0,x]，此时期望的重复次数为

g(k)={1,k≤d1+f(k),k>dg(k) = \begin{cases} 1, & k \leq d \\ 1 + f(k), & k > d \end{cases}g(k)={1,1+f(k),​k≤dk>d​

由全期望公式可知

f(x)=Eg(k)=1+1x⋅∫dxf(t)dtf(x) = Eg(k) = 1 + \frac{1}{x} \cdot \int_{d}^{x} f(t) \text{d} tf(x)=Eg(k)=1+x1​⋅∫dx​f(t)dt

解上述积分方程并代入初值条件得

f(x)=1+ln⁡xdf(x) = 1 + \ln \frac{x}{d}f(x)=1+lndx​

方差​

定义​

设随机变量 XXX 的期望 EXEXEX 存在且期望

E(X−EX)2E(X - EX)^2E(X−EX)2

也存在，则称上式的值为随机变量 XXX 的 方差，记作 DXDXDX 或 Var(x)Var(x)Var(x)。方差的算术平方根称为 标准差，记作 σ(X)=DX\sigma(X) = \sqrt{DX}σ(X)=DX​。

方差的性质​

若随机变量 XXX 的方差存在，则

  • 对任意常数 a,ba, ba,b 都有 D(aX+b)=a2⋅DXD(aX + b) = a^2 \cdot DXD(aX+b)=a2⋅DX
  • DX=E(X2)−(EX)2DX = E(X^2) - (EX)^2DX=E(X2)−(EX)2

协方差与相关系数​

一般来说，等式 D(X+Y)=DX+DYD(X + Y) = DX + DYD(X+Y)=DX+DY 并不成立，我们自然会提出两个问题：

  • D(X+Y)D(X + Y)D(X+Y) 与 DX+DYDX + DYDX+DY 之间相差的部分到底是什么。
  • D(X+Y)D(X + Y)D(X+Y) 与 DX+DYDX + DYDX+DY 在什么情况下相等。

对于第一个问题，我们引入协方差作为解答。

协方差的定义​

对于随机变量 X,YX, YX,Y，称

E((X−EX)(Y−EY))E((X - EX)(Y - EY))E((X−EX)(Y−EY))

为 XXX 与 YYY 的 协方差，记作 Cov⁡(X,Y)\operatorname{Cov}(X, Y)Cov(X,Y)。

协方差的性质​

对于随机变量 X,Y,ZX, Y, ZX,Y,Z，有

  • Cov⁡(X,Y)=Cov⁡(Y,X)\operatorname{Cov}(X, Y) = \operatorname{Cov}(Y, X)Cov(X,Y)=Cov(Y,X)
  • 对任意常数 a,ba, ba,b，有 Cov⁡(aX+bY,Z)=a⋅Cov⁡(X,Z)+b⋅Cov⁡(Y,Z)\operatorname{Cov}(aX + bY, Z) = a \cdot \operatorname{Cov}(X, Z) + b \cdot \operatorname{Cov}(Y, Z)Cov(aX+bY,Z)=a⋅Cov(X,Z)+b⋅Cov(Y,Z)

同时协方差与方差也有如下联系：

  • DX=Cov⁡(X,X)DX = \operatorname{Cov}(X, X)DX=Cov(X,X)
  • D(X+Y)=DX+2Cov⁡(X,Y)+DYD(X + Y) = DX + 2 \operatorname{Cov}(X, Y) + DYD(X+Y)=DX+2Cov(X,Y)+DY
关于协方差

你可能会发现协方差的性质与向量内积的运算性质在形式上高度一致。

在泛函分析的视角下，对于给定的概率空间，其上的全体随机变量构成一个线性空间，而协方差是这个空间上的一个内积，标准差则是由该内积导出的范数。

对于刚才提出的第二个问题，不难看出 D(X+Y)=DX+DYD(X + Y) = DX + DYD(X+Y)=DX+DY 当且仅当 Cov⁡(X,Y)=0\operatorname{Cov}(X, Y) = 0Cov(X,Y)=0。一个直观的必要条件是 XXX 与 YYY 独立，因为此时有

Cov⁡(X,Y)=E((X−EX)(Y−EY))=E(X−EX)E(Y−EY)=0\operatorname{Cov}(X, Y) = E((X - EX)(Y - EY)) = E(X - EX) E(Y - EY) = 0Cov(X,Y)=E((X−EX)(Y−EY))=E(X−EX)E(Y−EY)=0

但这个条件并不是充分的。为了描述满足 Cov⁡(X,Y)=0\operatorname{Cov}(X, Y) = 0Cov(X,Y)=0 的随机变量 XXX,YYY 之间的关系，我们引入相关系数

相关系数​

对于随机变量 X,YX, YX,Y，称

Cov⁡(X,Y)σ(X)σ(Y)\frac{ \operatorname{Cov}(X, Y)}{ \sigma(X)\sigma(Y) }σ(X)σ(Y)Cov(X,Y)​

为 XXX 与 YYY 的 Pearson 相关系数，记作 ρX,Y\rho_{X,Y}ρX,Y​。

Pearson 相关系数描述了两个随机变量之间线性关联的紧密程度。∣ρX,Y∣|\rho_{X,Y}|∣ρX,Y​∣ 越大，则 XXX 与 YYY 之间的线性关联程度越强。不难证明 ∣ρX,Y∣≤1|\rho_{X,Y}| \leq 1∣ρX,Y​∣≤1，且 ∣ρX,Y∣=1|\rho_{X,Y}| = 1∣ρX,Y​∣=1 仅可能出现在以下两种情况

  • 当存在实数 aaa 和正实数 bbb 使得 P(X=a+bY)=1P(X = a + bY) = 1P(X=a+bY)=1 时，有 ρX,Y=1\rho_{X,Y} = 1ρX,Y​=1；
  • 当存在实数 aaa 和负实数 bbb 使得 P(X=a+bY)=1P(X = a + bY) = 1P(X=a+bY)=1 时，有 ρX,Y=−1\rho_{X,Y} = -1ρX,Y​=−1。

当 ρX,Y=0\rho_{X,Y} = 0ρX,Y​=0 时我们称随机变量 XXX 与 YYY 不相关，此时 XXX 和 YYY 之间不存在线性关系。

「不相关」与「独立」

两随机变量不相关只是表明他们之间没有线性关联，并不代表没有其他形式的联系。

因此两随机变量 X,YX, YX,Y 不相关是他们相互独立的 必要而不充分 条件。

对于这一小节开头提到的第二个问题，我们给出结论：Cov⁡(X,Y)=0\operatorname{Cov}(X, Y) = 0Cov(X,Y)=0 的充要条件就是 XXX,YYY 中的某一个以概率 111 取常值，或 X,YX, YX,Y 不相关。
