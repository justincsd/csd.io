随机变量

相关概念​

随机变量​

给定概率空间 (Ω,F,P)(\Omega, \mathcal{F}, P)(Ω,F,P)，定义在样本空间 Ω\OmegaΩ 上的函数 X:Ω→RX : \Omega \to \mathbb{R}X:Ω→R 若满足：对任意 t∈Rt \in \mathbb{R}t∈R 都有

{ω∈Ω:X(ω)≤t}∈F\{ \omega \in \Omega : X(\omega) \le t \} \in \mathcal{F}{ω∈Ω:X(ω)≤t}∈F

则称 XXX 为 随机变量。

示性函数​

对于样本空间 Ω\OmegaΩ 上的事件 AAA，定义随机变量

IA(ω)={1,ω∈A0,ω∉AI_A(\omega) = \begin{cases} 1, & \omega \in A \\ 0, & \omega \notin A \end{cases}IA​(ω)={1,0,​ω∈Aω∈/A​

称 IAI_AIA​ 是事件 AAA 的 示性函数。

分布函数​

对于随机变量 XXX，称函数

F(x)=P(X≤x)F(x) = P( X \leq x )F(x)=P(X≤x)

为随机变量 XXX 的 分布函数。记作 X∼F(x)X \sim F(x)X∼F(x)。

分布函数具有以下性质：

  • 右连续性：F(x)=F(x+0)F(x) = F(x + 0)F(x)=F(x+0)
  • 单调性：在 R\mathbb{R}R 上单调递增（非严格）
  • F(−∞)=0F(-\infty) = 0F(−∞)=0,F(+∞)=1F(+\infty) = 1F(+∞)=1

同时我们可以证明，满足上述要求的函数都是某个随机变量的分布函数。因此，分布函数与随机变量之间一一对应。

随机变量的分类​

随机变量按其值域（根据定义，随机变量是一个函数）是否可数分为 离散型 和 连续型 两种。

离散型随机变量​

设 XXX 为离散型随机变量，其所有可能的取值为 x1,x2,⋯x_1, x_2, \cdotsx1​,x2​,⋯，则我们可以用一系列形如 P{X=xi}=piP\{ X = x_i \} = p_iP{X=xi​}=pi​ 的等式来描述 XXX。这就是我们在高中课本中学过的 分布列。

连续型随机变量​

设 XXX 为连续型随机变量，考察 P{X=x}P\{ X = x \}P{X=x} 往往是无意义的（因为这一概率很可能是 000）。

为什么说概率「很可能」是 000

考虑这样的随机变量 XXX：它以 12\frac{1}{2}21​ 的概率取 000，以 12\frac{1}{2}21​ 的概率服从开区间 (0,1)(0, 1)(0,1) 上的均匀分布。显然 XXX 满足连续型随机变量的定义。

对任何实数 r∈(0,1)r \in (0, 1)r∈(0,1)，不难得到 P{X=r}=0P\{ X = r \} = 0P{X=r}=0，但同时有 P{X=0}=12P\{ X = 0 \} = \frac{1}{2}P{X=0}=21​。

另一方面，设 X∼F(x)X \sim F(x)X∼F(x)，则

P(l<x≤l+Δx)=F(l+Δx)−F(l)P( l < x \leq l + \Delta x ) = F(l + \Delta x) - F(l)P(l<x≤l+Δx)=F(l+Δx)−F(l)

一个自然的想法是用极限 lim⁡Δx→0+F(l+Δx)−F(l)Δx\lim\limits_{\Delta x \to 0^+} \frac{F(l + \Delta x) - F(l)}{\Delta x}Δx→0+lim​ΔxF(l+Δx)−F(l)​ 来描述 XXX 取值为 lll 的可能性。

这个式子就是我们熟知的导数，于是问题转化为寻找一个非负函数 f(x)f(x)f(x) 使得

F(x)=∫−∞xf(x)dxF(x) = \int_{-\infty}^{x} f(x) \text{d} xF(x)=∫−∞x​f(x)dx

若这样的 f(x)f(x)f(x) 存在，则称之为 XXX 的 密度函数。

随机变量的独立性​

前面讨论了随机事件的独立性。由于随机变量和随机事件紧密联系，我们还可以类似地给出随机变量独立性的定义。

定义​

若随机变量 X,YX, YX,Y 满足对任意的 x,y∈Rx, y \in \mathbb{R}x,y∈R 都有

P(X≤x,Y≤y)=P(X≤x)P(Y≤y)P( X \leq x, Y \leq y ) = P( X \leq x ) P( Y \leq y )P(X≤x,Y≤y)=P(X≤x)P(Y≤y)

则称随机变量 X,YX, YX,Y 独立。

Note

注意到，中学课本中对随机变量独立性的定义是用形如 P(X=α)P(X = \alpha)P(X=α) 的概率定义的，但由于连续性随机变量取特定值的概率通常是 000，故在更一般的情形下借助分布函数定义才是更加明智的选择。

性质​

若随机变量 XXX,YYY 相互独立，则对于任意函数 f,gf, gf,g，随机变量 f(X)f(X)f(X) 与 g(Y)g(Y)g(Y) 相互独立。

注意

有时候我们会研究相互独立的随机变量 XXX,YYY 的某一函数 f(X,Y)f(X, Y)f(X,Y)（如 XY2XY^2XY2）的分布。

尽管 XXX 与 YYY 是独立的，但不能想当然地认为对 YYY 的某一取值 yyy，f(X,y)f(X, y)f(X,y) 与 f(X,Y)f(X, Y)f(X,Y) 服从同样的分布。
